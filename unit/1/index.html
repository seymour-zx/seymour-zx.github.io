记录与分享零基础自主学习HTML的经历

颐亲王  正协信息客栈
发布于 2020-11-29T10:39Z

为什么要建设自己的个人网站？

最直接的原因，就是突然感觉自己建设一个简单的网站，其实是不难实现的。

说实话，以前真的觉得很麻烦。即使各种重生现代都市的商业小说里，开局就来个架设导航网站的操作剧情。 看起来好像很容易复制，但设置服务器这些还是让我觉得很麻烦，打消了学习建站的念头。

2020年10月初，无意中看到了很多培训机构的招生广告，像什么“一天入门，一个星期掌握，一个月成神”，“进群免费学”之类的。

无论硬广还是软广，广告看多了，不禁要问自己，对于像我这种HTML零基础的新人来说，自建网站的愿望，到底难不难实现？怎么感觉好像“我也行！”？

为什么选择"zhengxie.info"这个域名？

怎么说呢，都是心血来潮的结果。查过域名注册信息的人肯定知道， "zhengxie.info" 这个域名是我在北京时间凌晨2点后通过"阿里云"注册的。

"zhengxie"是姓名的全拼，"zhengxie"这个字段还没被注册的域名真不多。".info"这个后缀，就是在尚可注册的域名里，挑选一个比较适合个人使用的。

中意域名的可选余地不大，但还能从中捡漏到"zhengxie.info"，我是感觉无比幸运的，因为我越来越喜欢这个域名。

注册到心仪的域名，才更有动力去好好地经营自己的网站。

一开始，我是在腾讯云选域名的，也是打算在腾讯云注册域名。但我现在也忘记为什么当时会选着选着，跑到阿里云选择域名，然后下载阿里云app注册了域名。

域名注册需要实名认证审核，通常需要3-5个工作日。我的域名注册代理商是阿里云（万网）。在挑选域名期间，可以先提交资料，弄一个实名信息认证的模板，审核也是需要3-5个工作日。购买域名时可以使用认证过的信息模板，可以省去等待时间。

我当时购买时，没使用模板。购买后认证时，让我先认证模板。等了三个工作日后，又让我等待域名实名认证，前后等了一个星期。可能我把顺序弄错，多等了三个工作日。但我不确定能否不认证模板，直接实名认证域名。

注册GitHub账号, 使用GitHub Pages服务

注册完成后登录账号，网页界面右上角区域，点击+标，下拉选单第一行"New repository"。

在"Repository name *"，按双引号内的各式填写：“所有者名称.github.io”

勾选"Add a README file"

再点击绿色按钮"Create repository"

页面跳转后，点击右边"Settings"，没看到"Settings"就先点"…"

下滑到"GitHub Pages", "Your site is published at "后就是你的网页了，（记住这个网址，下一步解析是用到）

鼠标滚轮再下滑，"Custom domain"，在框内填写你注册的域名，如"zhengxie.info"，然后点击"Save"完成保存。

等待审核好几天，终于可以解析域名了

我是用阿里云注册域名的，这里用阿里云举例如何解析域名。

打开阿里云App，"管控"界面，下滑到"我的云产品"，点击"全部展开"

选择"云解析DNS"，在点击域名会自动跳到"解析列表"

点击屏幕右上角"+"，记录类型选择"CNAME"

主机记录：如果填@就是解析到本级域名，如https://zhengxie.info/ ，如果填"www"就是解析到相应的子级域名，如https://www.zhengxie.info/ 。

记录值：把GitHub给的地址填进去，（就是把上一步让记住的网址填进去）。

等10分钟左右，就能访问指定的域名了
